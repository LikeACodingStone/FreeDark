# 内置图标 QIcon
```
	//使用示例
	QPushButton btn("TEST");
	QStyle *style = QApplication::style();
	QIcon icon = style->standardIcon(QStyle::SP_DriveCDIcon);
	btn.setIcon(icon);
```
