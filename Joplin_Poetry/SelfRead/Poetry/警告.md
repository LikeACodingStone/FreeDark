# 警告 2022/6
```
终将会有一支历史之笔。
记录下你们今天的疯狂。
终将会有一天。
有人审视你们过往。
所有的那些荒诞、扭曲、肆无忌惮的野蛮。
并且以此为荣光或者无声默认的准则。
终究会随着这场运动的结束而终结。
也就表明，你现有的权力终将被没收。
而你们用沾满罪恶的双手开辟一个丑陋的时代却远远不那么轻易结束。
你们用残忍埋下一颗颗仇恨的种子。
一颗颗布满荆棘的藤蔓。
终有一日，终有一时，
这一颗颗种子会包裹在你的周围。
并且生根发芽成长。
这一颗颗藤蔓要将你牢牢困死。
万一就算万一，
你侥幸的能躲的了这场灾难。
你的子子孙子也将大概率替你偿还。
被你们荼毒过的子子孙孙会跟你今天一样丧心病狂。
会毫不犹豫的清算你们那些正义的过往。
而你们。
必将背负昭彰臭名。
带着这份荣耀下葬。
如果审判存在的话。
死后还将万劫不复、永不翻身。
同而为人，我替你们感到羞耻。
同而为人，我替自己所处的时代感到罪恶。
后入会不断指着这一批次的墓碑说。
我们就是摧毁人类的文明的刽子手。
是铁链枷锁的守护神。
是自相屠杀的罪恶之徒。
```
