# 失眠马拉松  2017/8
```
再过两百三十九分钟
三十八、三十七分钟
闹钟就会响起
你像往常一样
人模狗样的走出这扇十四平米的房门
地铁公交、早餐地铁
就三个小时吧
你说睡觉这种事
又不是自由落体
该是多久就是多久
困了自然就困了
困着困着就睡了

忽然一只蚊虫刺破平静
在耳角后背狠狠插进肉里
从惊醒到清醒
火烧一样的疼
你怀揣最大仇恨发誓必须弄死它
就像一个虔诚的邪教徒
无比坚定

一场血淋淋屠杀之后
空荡荡的黑夜里填满了空荡荡的躯壳
寂寞已久的灵魂四处猎艳着宣泄的出口
那个D罩杯的女护士睡着了吗
那个前凸后翘的公司前台睡着了吗
那个瓜子脸的老板娘睡着了吗
那个00后的女大学生睡着了吗
她在哪里对着镜头抚慰
她在哪里手握黄瓜窃喜
她在哪里被第四次摇醒
她又在哪里哭泣、呻吟
此刻被蚊子咬过的男人正当亢奋
而那只五十块钱买的淘宝灭蚊灯早已罢工
他在这里手握百年孤独
像失眠了一百年一样
开始新的一天
```
