# 一段忧伤的过往  2020/11

```
那堆扬起灰尘的信物
信封上的笑脸
都是表演的工具
所有的落叶都已经远去
像那些所有过往的记忆
有的被焚烧进了大气
有的埋进泥土里
你耗尽岁月与光阴
一事无成
为情所困
如果不是一场意外的冲刷
显露出那些本该的痕迹
也许你还停留在
一场臆想的梦境里
你自以为阅历过遍地人渣
逃脱了时代的命运
终究发现不过自以为是
你像所有你看不上的傻逼一样
干着一样的勾当
真诚、卑微而又廉价
而立之年
美好的过往化作云烟
生命的过往被一步步残食
逐渐失去回忆的勇气
过去的伤疤越长越深
以至于再也不能去触碰一下
曾经所有那些美好的一切
而今变得如此面目可憎
你试着去理解这一切
试着用逻辑去分析这一切
说服自己把这一切都合理化
你挤尽了智慧和气力去回溯
感到胸前一阵剧烈的疼痛
你摸了摸疼痛的位置
左心房胸口正中央
那一块空空如也
如同大气一样冰凉
像泥土一样潮湿
```
