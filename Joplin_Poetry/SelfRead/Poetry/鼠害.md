# 鼠害  2022/1
```
假如我是一块镜头
我必定能定格他们随机散落的地点
必定能捕捉他们茫然四顾的神情
像是大自然刚出土的一窝煤炭工人
一窝五到八只不等
假如我是一只麦克风
我必定能记录这虚伪的喧嚣
必定分离出他们窃窃私语的阴谋
必定撕毁那刻意制造的自然假象
像是踩死粪堆上刚爬出来的一窝屎壳郎
然而我只是一具肉体
困在这嘈杂的检票口前
任由他们观察我、拦住我、摸我
假装什么也没有发生过
然后不回头的继续往前走
走过轰鸣的铁轨
让列车长驱直入的碾压
我甚至不用思考这么做的理由
就像所有的行人一样沉默、坦然并且麻木
都不需要完全睁开眼就能看清楚他们数量越来越多
因为那一条条慵懒的表情正在昭告天下
他们早就给反叛者准备好充足的陷阱
于是枪杀蛀虫已然毫无意义
一坨坨正在破壳的赤蛆
不停从造粪机器向外喷涌
络绎不绝
就在不久前
我还被那迷美肉体下清澈的瞳孔击中
那来自可见的浅薄跟单纯世界
然而她毫无疑问的没过太久
就如同枯草一般黯然熄灭
就好像几千年来一遍遍上演的还魂戏
```
