# 我唯一在坚持的就是等死 2022/10/25
```
我唯一在坚持的就是等死
等待被突如其来所杀死
或者被突如其来所激发
拉个把人陪我去死
我未救赎的肉体
恐惧疼痛，饥饿与疲劳
我不敢也不愿轻易直起腰来
我唯一能做的就是等死
等着自己亦步亦趋的死
当然也包括把他熬死
当然最好是他先死
我用尽一个卑鄙之人有限的气力
铁锤从眼前落地
改变不了半毫米轨迹
我唯一能做的是看着他死
假如我不能心安理得说服自己
可能会需要陪他去死
不要问我最近怎么样了
在我理解里
这句话来源自于生活
而现在生活的墓碑四处林立
虽然活着的人还一如既往的
睡的很多 吃的也不少
然而这唯一的坚持只是在等死
等着被人杀死
或者把人熬死
熬到双眼红肿
血丝爬满天安门城楼
熬到口干舌燥
烈火燃尽了水晶棺

```
