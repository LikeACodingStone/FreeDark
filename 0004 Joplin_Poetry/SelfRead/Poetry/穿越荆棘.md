# 穿越荆棘 2020/10
```
从上路已经走过太久
早已忘记轻盈自在的感觉
拄着我这双还能撑起身躯的双脚
蝗虫大的汗珠趟过脸颊
那一阵阵火辣
是刺条新划的开皮肉在反抗

刺啦刺啦血流经旧伤口
二十七条铁链穿透背骨
我挣扎着蹭开眼皮
早已明白
放空气力就被拉回出发的地方
是我生长的地方
专吃人头的地方
那里有我的孝子贤孙
那里有我的社会公仆
那里的人都顶着鸡蛋一样大的脑骸
蛋壳里装的是绿豆大的黄
是他们所有的精神食梁
向前奋进
疼痛拉开我的皮囊
感觉头颅在生长
脑袋里开始填满血肉
我变成一颗鲜活的头颅
我化成一泡乳色的脑浆
烈日蒸发了我
我在云层中俯视大地
镂空的脑壳们像一个大峰窝
我跟着雨水一起落下
让他们的头颅生长
让他们血肉伸张
```
