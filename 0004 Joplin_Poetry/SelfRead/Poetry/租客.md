# 租客  2021/9
```
我是一个客人
带着我的全家住到主人家里
一住就是一年
主人靠吸我的血活着
有时候也吸我家人的血
好在我的凝血功能还算健全
以至于做客七八年了
血液还源源不断生长
当然也不是完好如初
不同的主家各有所好
有些口味的主人我已经供应不了了

这些年我制造了恶臭的血
我吃完卫生间谁也不收的纸篓
吃完走廊跟过道外卖的垃圾袋
抹一抹嘴 黑色的恶臭喷涌而出

我还制造了下贱的血
我吃完东西南北四道墙的空心木材
然后吃掉上下铺之间的十公分
我吃完成年人的最后一抹尊严
抹一抹嘴 散发出下贱的酸味

我还制造了欺骗的血
我吃完整整十三页的贷款合同
然后吃掉销售员面不改色的良心
抹一抹嘴 谎言完好的盖住底线

我甚至制造了白色的血
我吃完家具和墙壁的甲醛
然后吃掉无孔不入的雾霾
抹一抹嘴 白血病的子女四处逃窜

```
