# 无名时刻  2020/4
```
将我丢弃路旁
僵硬冰凉又不埋葬
漫天的风雪四面八方
雷电在嫩绿的枝头围剿
寒流渗透满城药味的大街小巷
一个老妪坐在摩天楼顶上
搂着自己的遗像
俯身坠下
四周空空荡荡
没有一句独白、悲伤
有个诗人说
如果我死了
希望埋在家乡
他不知道此时此地
一切无处埋葬

我醒来穿过窗台
殷红的斜阳
映照着腐烂的空气
女人和孩子低声啜泣
沾满血迹的铁棍在一旁张牙舞爪
正好两天前
有人用它打死了十五年的老狗
因为有另一些人说
所有的狗都有几率传染病毒

恐惧沿着头皮上爬
高挂沉默四十年的扬声器
勃起后硬朗如初
村长操着方言问候路人全家
大部分行者不再说话
另一些时而大声喧哗
他们因懦弱变得沉默
他们因沉默已经癫狂
从此再也无人歌唱
道德与秩序在骤然改写
高尚者只有跟卑鄙者一样
才能存活
哪怕不能存活
也只能共用一块坟墓
共掩一堆黄土、一丘土坡
此时此地
无论贫富贵贱
均无名无氏
无人记得
```
