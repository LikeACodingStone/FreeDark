# 沙尘暴  2021/4
```
就像大地突如其来的失落情绪
就像黑白电视突然跳闪的雪花屏
就像光明努力穿不透的白内障
阳光下视线开始浑浊不清
假如你小时候捅过马蜂窝
准能料想得到
此番袭击必然是一次暴力革命
尘土卷起来落叶御风前进
诡异、妖娆又迅猛
迅速占领了每一寸在不久前
还属于柳絮跟白杨的领地
就像在二战之间
被德意志和苏联先后攻陷的波兰
呼吸道也在尘土跟柳絮之间
遭受着无声的渗透殖民
一缕昏黄将天空和大地
紧密连接在一起
像干柴烈火的孤男欲女
经过一阵热烈的短暂亲密
交媾的分泌物铺洒在了
大地之上每一寸肌肤
不论围墙建筑、山川还是草木
等到分离之后
沧桑感简直称得上
被上百年历史翻阅的遗迹
来往不知去向屎壳郎
忙忙碌碌穿行在四通八达的城市干线
他们时而双眼放光
时而高声鸣笛
```
