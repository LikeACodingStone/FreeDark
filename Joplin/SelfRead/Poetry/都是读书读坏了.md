# 都是读书读坏了  2023/1/23
```
如果不读书
我应该做一个农民工的儿子
三十而立
进城赚钱，回村结婚

可我读书读坏了
现在依然是农民工的儿子
城市和村庄我眼里成为一种群体现象
这片土地的人和事
成了一种文化特征
而在书里我
见过成百上千种文化特征
见过沧海桑田与时事变迁
见过丑陋的现实和荒诞的人生
见过那些看不见却发生过
见过那些发生过
而经历过的人都说不清楚的事情
这一切 都是读书读来的

可我读书读坏了
忘记了我是一名二代农民工
进城赚钱时
习惯了靠能力赢得尊重
习惯了靠想法获得认同
回村以后
总要挤着脑壳上的皱纹才想起
那些长辈教过我的东西
总要借助不断提醒自己才明白
精神世界和现实生活
如此大的剥离
以至于
一个在云中
一个在海里
```
